# 🏋️‍♂️ Fitness AI Coach & Workout Analytics Dashboard

[English Version Below]

منصة رقمية ذكية ومطورة باستخدام لغة بايثون ومكتبة **Streamlit** تعمل كمساعد ومدرب رياضي ذكي يعتمد على البيانات. يتيح التطبيق للمستخدمين تخطيط جداولهم الرياضية، حساب السعرات الحرارية المستهدفة، وتحليل أدائهم البدني عبر واجهة مستخدم سحابية ممتازة ومريحة للعين (SaaS Theme).

## 🚀 المميزات الأساسية (Core Features)
* **حسابات الـ Macros الذكية:** حساب فوري للسعرات الحرارية وتوزيع البروتين، الكربوهيدرات، والدهون بناءً على وزن المستخدم وهدفه البدني.
* **تخطيط الجداول الرياضية:** واجهة مرنة لتصميم وتتبع التمارين اليومية وشدة التدريب.
* **تحليل البيانات البصري:** رسومات بيانية تفاعلية لمراقبة تطور الوزن وحرق السعرات الحرارية مع الوقت.
* **مؤشرات الأداء الرياضي (KPIs):** كروت عرض تنفيذية ناعمة ومنسقة تلخص الحالة البدنية الحالية للمستخدم لتفادي تداخل البيانات.
